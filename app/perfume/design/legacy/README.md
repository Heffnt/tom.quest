# Legacy

`index.html` is the original standalone brewing bench (self-contained, opens by
double-click). **Retired 2026-07**: its embedded data blob and matching engine
predate the confirmed rules — no frequency combination, no brew multiples, the
old "craftable if excess ≤ ⊖ and missing ≤ ⊕" rule, and the invented flavor
texts. The live app at tom.quest/perfume (engine shared client/server) is the
one implementation of the system. Kept for reference only; not maintained.
